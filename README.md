## Environment
```dotenv
# 데이터베이스 연결 정보
DATABASE_URL=jdbc:mariadb://localhost:3306/db
DATABASE_USERNAME=
DATABASE_PASSWORD=

# JWT 암호화를 위한 키
JWT_ACCESS_SECRET=
JWT_REFRESH_SECRET=

# CORS 도메인 정보
CORS_HOST_DEVELOPMENT=http://localhost:5175
CORS_HOST_PRODUCTION=

# 소셜로그인 API 정보
NAVER_CLIENT_ID=
NAVER_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_SECRET=
KAKAO_CLIENT_ID=
KAKAO_SECRET=
```
